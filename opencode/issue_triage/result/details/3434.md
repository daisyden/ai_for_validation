## Root Cause

Running speech_transformer torchbench in inductor mode with amp_bf16 segfaults on XPU after a
Tensor.item() graph break. Maintainers bisected the regression to upstream pytorch/pytorch#177158
(Dynamo/Inductor change), and a fix has been posted as pytorch/pytorch#181515. The crash is a
SIGSEGV in the native layer triggered by partial-graph compile/exec after the graph break, not a
torch-xpu-ops kernel bug.

## Fix Approach

- Cherry-pick upstream fix PR pytorch/pytorch#181515 into release/2.12.
- verify torchbench speech_transformer amp_bf16 inference no longer segfaults.
- Track via the existing related issue #2057.
