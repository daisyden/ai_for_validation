## Root Cause

hf_T5/hf_T5_base inference inductor latency on BMG B60 regresses ~30-33% in PyTorch 2.12 vs 2.11
across both amp_bf16/amp_fp16, while eager latency is flat (<0.2%). Sole moving part across releases
is triton-xpu 3.7.0 -> 3.7.1, so regression is in inductor-codegen / triton-xpu kernel scheduling
for T5-style fused attention/MLP kernels on Xe2 (BMG).
