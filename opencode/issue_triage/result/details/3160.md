## Root Cause

All listed cases fail on Windows with 'InductorError: Failed to find C compiler. Please specify via
CC environment variable' raised from Triton's build path. This is a CI environment configuration
problem on Windows runners (no MSVC/cl on PATH or CC unset) when Inductor tries to compile cpp
wrapper / Triton host code. Maintainer confirmed same root cause as #2660 and the issue is already
CLOSED.
