## Root Cause

test_schema_check / test_meta on aten::_flash_attention_forward fails with NotImplementedError
because there is no XPU implementation registered for this op; PyTorch dispatch falls back to CPU
which also has no implementation (CUDA-only). The op is declared in
pytorch/aten/src/ATen/native/native_functions.yaml as CUDA-only and torch-xpu-ops has no kernel
under src/ATen/native/transformers/sycl/. Duplicate of #2285.

## Fix Approach

- Either (a) register an XPU kernel for _flash_attention_forward in torch-xpu-ops (transformers/sycl/) backed by oneDNN SDPA or a custom SYCL flash kernel, or (b) update the upstream OpInfo/skip list in pytorch to mark _flash_attention_forward as CUDA-only so the schema/meta tests are gated.
- Track via duplicate #2285.
