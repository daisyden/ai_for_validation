## Root Cause

test_distributed_spawn.py::TestDistBackendWithSpawn::test_DistributedDataParallel fails with
BACKEND=xccl WORLD_SIZE=4: child process exits with code 10 during DDP forward/backward, indicating
accuracy/collective mismatch in ProcessGroupXCCL allreduce path used by DDP gradient reduction. Test
is being enabled on the madhumitha0102/distributed_2.10 branch as part of 2.10 distributed
enablement; XCCL allreduce semantics or DDP bucket flattening differ from CUDA/NCCL reference.

- Likely needs a fix in torch-xpu-ops/src/xccl/ProcessGroupXCCL.cpp or upstream DDP reducer when xccl backend is selected.
