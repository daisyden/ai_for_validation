## Root Cause

Feature request: torch.xpu.get_device_properties() in torch/xpu/__init__.py exposes
name/total_memory/etc. but no field distinguishing iGPU vs dGPU. Maintainer guangyey confirms this
requires a SYCL extension query (e.g., ext_intel_device_id / device.is_integrated()) which is not
yet wired through DeviceProp in torch/csrc/xpu/Module.cpp.

## Fix Approach

- Extend XPU DeviceProp struct in torch/csrc/xpu/Module.cpp and aten/src/ATen/xpu/XPUContext.cpp to include a device_type/is_integrated field populated via SYCL aspect::ext_intel_device_id or device.get_info<info::device::host_unified_memory>().
- expose through torch.xpu.get_device_properties().
