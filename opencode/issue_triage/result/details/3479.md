## Root Cause

Five upstream profiler tests (TestProfiler.test_basic_chrome_trace,
test_disable_external_correlation, test_dynamic_toggle,
TestExperimentalUtils.test_utils_get_optimizable_events, test_profiler_pattern_matcher_json_report)
fail under XPU. test_basic_chrome_trace asserts the chrome trace metadata event name == 'PyTorch
Profiler (0)' (test_profiler_xpu.py:2116) but the XPU profiler emits a different name/format, and
the helper _validate_basic_json was not adapted for the XPU device branch. Per maintainer comment
from @PawelSwider2000, most are test-side fixes pending in PR #3493; test_dynamic_toggle and
test_disable_external_correlation are deeper Kineto-side bugs in XPU correlation/toggle handling.
