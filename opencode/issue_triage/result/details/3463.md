ref 0.00058 (3x multiplier). Eager path is unaffected; failure is in inductor-compiled bf16 path.
Owner-tagged duplicate of long-standing issue intel/torch-xpu-ops#1811 (Triton-related bf16 numeric
drift in benchmarks/dynamo on PVC). Reproduces only with torch 2.12 RC + triton-xpu 3.7.1.
