## Root Cause

Eight cases in third_party/torch-xpu-ops/test/xpu/dynamo/test_misc_xpu.py drift from the torch
nightly wheel 2.13.0.dev20260422+xpu. Sub-cause 1 (test_compile_non_infra_*) raises 'fullgraph=True
found no compiled frames' from torch/_dynamo/eval_frame.py compile_wrapper which lacks an XPU-aware
path; sub-cause 2 (deepcopy/duplicate_graph_break/cache_invalidate) shows compile-counter drift on
XPU; sub-cause 3 is wording drift in torch._check; sub-cause 4
(test_nested_function_resuming_with_correct_globals) is a port bug — utils.py shim is missing under
test/xpu/dynamo/ and sys.path does not include upstream test/dynamo, so 'from .utils import
outer_func' fails with ModuleNotFoundError.
