## Root Cause

Whisper-tiny under --amp --amp-dtype bfloat16 inference fails with Dynamo dtype-mismatch
(torch/_dynamo/utils.py:3365: 'dtype mismatch torch.float16, torch.bfloat16'). The model has fp16
constants/buffers that collide with the bf16 amp autocast region, so guarded fake-tensor propagation
diverges. Fix tracked in upstream PR pytorch/pytorch#180309; verified passing by mengfei25.
