## Root Cause

test/distributed/checkpoint/test_state_dict_stager.py::test_tensor_pinned_and_shared expects the
staged tensor's `.is_pinned()` to be True after staging with pin_memory=True, but on XPU it returns
False. The DCP stager pin_memory path does not yet pin XPU host memory correctly (likely uses
CUDA-specific pin_memory APIs in torch.distributed.checkpoint stager utilities, or XPU's is_pinned()
reporting is incorrect on the staged storage).
