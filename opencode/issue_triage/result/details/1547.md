## Root Cause

NotImplementedError: `symm_mem::fused_matmul_reduce_scatter` has no XPU implementation. The op is
registered in pytorch/torch/distributed/_symmetric_memory/__init__.py for CUDA only; XPU symmetric
memory backend is missing. Maintainer (Chao1Han) confirmed dependency on symm_mem support, same root
cause as #1551/#1548.

## Fix Approach

- Add XPU symm_mem backend and register XPU kernel for `symm_mem::fused_matmul_reduce_scatter` (composing oneCCL reduce_scatter with oneDNN matmul).
- Add tests to test/xpu/skip_list_dist*.py until implementation lands.
