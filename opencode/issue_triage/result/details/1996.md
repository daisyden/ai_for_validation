Feature request to enable TorchAO memory-efficient optimizers
(https://github.com/pytorch/ao/tree/main#memory-efficient-optimizers) on XPU. The prerequisite XPU
memory-trace feature was completed (pytorch/pytorch#168262, xpu-ops #2311). The remaining enablement
work (pytorch/ao#3191) is still in progress and tracked internally in PYTORCHDGQ-8060.

- No code change is required in torch-xpu-ops itself until TorchAO exposes the XPU code path.
