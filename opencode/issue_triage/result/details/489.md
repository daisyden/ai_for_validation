## Root Cause

torchbench Moco model raises NotImplementedError('xpu not supported') at
benchmark/torchbenchmark/models/moco/__init__.py:80 because the model script only whitelists 'cuda'
and additionally requires DDP/NCCL setup. Failure occurs at model load before any aten op runs, so
it is an upstream torchbench/benchmark-harness gap (compounded by needing XCCL DDP support). Fix PR
pytorch/benchmark#2616 was filed to enable xpu device.

## Fix Approach

- Merge pytorch/benchmark PR #2616 to add xpu branch in moco/__init__.py and switch the DDP backend to xccl when device==xpu.
- bump torchbench pin in pytorch/pytorch test-infra.
- Verify ProcessGroupXCCL DDP path is functional before re-enabling moco in the XPU torchbench accuracy suite.
